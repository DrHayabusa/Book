Series Rules & Formulas

Parallel Circuits
Parallel Rules & Formulas
⚡ Same voltage across every branch: V₁ = V₂ = V₃ = V
🔀 Currents add up: I_T = I₁ + I₂ + I₃
🔧 Reciprocal resistance: 1/R_T = 1/R₁ + 1/R₂
1/R_T = 1/R₁ + 1/R₂ + ... + 1/Rₙ
R_T = (R₁ × R₂)/(R₁ + R₂)2 resistors only
I_T = I₁ + I₂ + ... + Iₙ
Remember: R_T is always less than smallest resistor

Kirchhoff's Laws
KCL – Current Law (at a node)
ΣI_in = ΣI_out
Sum of all currents entering a node = Sum leaving
KVL – Voltage Law (around a loop)
ΣV = 0
Sum of all voltage rises & drops around any closed loop = 0
+V_s − V₁ − V₂ − V₃ = 0  →  V_s = V₁ + V₂ + V₃

V₂ = V_s × R₂/(R₁+R₂)    V₁ = V_s × R₁/(R₁+R₂)

Thévenin's & Norton's Theorems
Thévenin's Theorem
Any linear circuit between two terminals = one voltage source (V_th) in series with one resistance (R_th)
STEP 1 – Find V_th
V_th = V_oc
Open-circuit voltage at terminals
STEP 2 – Find R_th
R_th = V_oc / I_sc
Kill sources: V→short, I→open
Norton's Theorem & Conversion
Any linear circuit = one current source (I_N) in parallel with one resistance (R_N = R_th)
I_N = I_scShort-circuit current
R_N = R_thSame resistance!
Thévenin → Norton
I_N = V_th / R_th
Norton → Thévenin
V_th = I_N × R_N
Maximum Power Transfer
R_L = R_th
P_max = V_th² / (4 R_th)
Maximum power delivered to load when load resistance equals Thévenin resistance
Superposition Principle
In a linear circuit with multiple sources, the response (V or I) at any point = sum of responses due to each source acting alone
• Kill other voltage sources → short circuit
• Kill other current sources → open circuit
• Sum all individual results
Source Transformation
V_s = I_s × R
I_s = V_s / R
Voltage source + series R ↔ Current source + parallel R
(Same external behaviour)
