1. Core Quantities You Must Know

If the exam starts with a basic definition or unit question, these are the first lines you should remember.

High Yield Definitions

Current (I)

Rate of flow of charge.

I = Q / t

Unit: ampere (A)

Voltage (V)

Energy per unit charge between two points.

V = W / Q

Unit: volt (V)

Resistance (R)

Opposition to current flow.

R = V / I

Unit: ohm (Omega)

Power (P)

Rate of electrical energy transfer.

P = V I

Unit: watt (W)

Charge and Energy

  • Q = I t
  • E = P t
  • E = V I t
  • 1 kWh = 3.6 x 10^6 J

Resistivity

For wire questions:

R = rho L / A
  • Longer wire -> larger R
  • Thicker wire -> smaller R

Sign Conventions

  • Across a resistor, voltage drops in direction of current.
  • When traversing a source from - to +, count a rise.
  • Choose a direction and stay consistent.

3. Series vs Parallel Circuits

This is the most common exam trap. Always identify whether the same current flows through components, or the same voltage appears across them.

Most Tested Idea

Series Circuit

R_total = R1 + R2 + R3 + ...
  • Current is the same through every component.
  • Voltage divides across components.
  • If one element breaks, the whole path breaks.
Think: one path only.

Parallel Circuit

1 / R_total = 1 / R1 + 1 / R2 + 1 / R3 + ...
  • Voltage is the same across each branch.
  • Current splits between branches.
  • Equivalent resistance is always less than the smallest branch resistor.
Think: multiple paths.

4. Ohm's Law, KCL, and KVL

These three ideas are the skeleton of circuit analysis. If a question feels messy, return to these laws first.

Solve From Here

Ohm's Law Triangle

  • Cover the unknown to rearrange quickly.
  • V on top, I and R at bottom.

KCL: Current Law

Sum of currents entering = Sum of currents leaving
I1 + I3 = I2 + I4

KVL: Voltage Law

Sum of rises - Sum of drops = 0
+Vs - I R1 - I R2 - ... = 0

5. Voltage Divider and Current Divider

Divider rules appear everywhere in fast circuit questions. Use them when resistors are clearly in series or clearly in parallel.

Shortcut Formulas

Voltage Divider Rule

Vout = Vs x (R2 / (R1 + R2))
  • Works when resistors are in series.
  • Output voltage is taken across the chosen resistor.
  • Bigger resistor in series -> bigger voltage drop.

Current Divider Rule

Current in one branch is inversely proportional to that branch resistance
For two resistors: I1 = Itotal x (R2 / (R1 + R2))
  • Smaller resistance gets larger current.
  • Use only for parallel branches.

6. Power, Energy, and Measuring Devices

Easy marks if units stay clean. These formulas also show up inside longer problems.

Units Matter

Power Relations

P = V I
P = I^2 R
P = V^2 / R
  • Use P = VI when both V and I are known.
  • Use P = I^2R in series-current questions.
  • Use P = V^2/R when voltage across resistor is known.

Energy and Cost

E = P t
1 Wh = 3600 J
1 kWh = 3.6 x 10^6 J
  • Convert hours to seconds if answer must be in joules.
  • Keep watts, volts, and amps consistent.

Ammeter vs Voltmeter

  • Ammeter: series connection, very low resistance.
  • Voltmeter: parallel connection, very high resistance.
  • Wrong connection changes the whole circuit.
A in series, V in parallel

7. Capacitors and Inductors

These components store energy, but they behave differently. A capacitor resists sudden voltage change. An inductor resists sudden current change.

Storage Components

Capacitor Cheat Sheet

Q = C V
i = C (dv/dt)
E = 1/2 C V^2
  • In parallel: Ctotal = C1 + C2 + ...
  • In series: 1/Ctotal = 1/C1 + 1/C2 + ...
  • At steady DC, ideal capacitor acts like an open circuit.

Inductor Cheat Sheet

v = L (di/dt)
E = 1/2 L I^2
  • In series: Ltotal = L1 + L2 + ...
  • In parallel: 1/Ltotal = 1/L1 + 1/L2 + ...
  • At steady DC, ideal inductor acts like a short circuit.

8. RC and RL Transients

If a graph or charging question appears, remember the time constant and the shape: exponential, not straight line.

Graph Questions

RC Charging and Discharging

tau = R C
Charging: Vc = Vs (1 - e^(-t/RC))
Discharging: Vc = V0 e^(-t/RC)
  • At 1tau, capacitor is about 63% charged.
  • At 5tau, it is practically complete.

RL Current Growth

tau = L / R
I(t) = Imax (1 - e^(-tR/L))
  • Inductor current cannot jump instantly.
  • At steady DC, inductor behaves like a short.

9. AC Basics, RMS, and Impedance

If the exam includes alternating current, remember peak, RMS, reactance, and phase difference.

AC Essentials

Waveform and RMS

Vrms = Vpeak / sqrt(2)
Irms = Ipeak / sqrt(2)
  • RMS is the DC-equivalent heating value.
  • Use RMS values in power calculations for AC resistive loads.

Reactance and Impedance

X_L = 2 pi f L
X_C = 1 / (2 pi f C)
Z = sqrt(R^2 + (X_L - X_C)^2)
  • Inductor: voltage leads current.
  • Capacitor: current leads voltage.
  • Pure resistor: voltage and current are in phase.
Power factor = cos(phi)

10. Source Transformation, Thevenin, Norton

These methods simplify ugly networks into one clean source and one clean resistance.

Network Simplification

Source Transformation

Voltage source Vs in series with Rs
is equivalent to
Current source Is = Vs / Rs in parallel with Rs
  • Use when it makes the network easier to combine.

Thevenin Equivalent

  1. Remove the load.
  2. Find open-circuit voltage -> Vth.
  3. Kill independent sources and find resistance seen from the terminals -> Rth.
Equivalent = Vth in series with Rth

Norton Equivalent

  1. Find short-circuit current -> In.
  2. Find equivalent resistance -> Rn = Rth.
Equivalent = In in parallel with Rn
In = Vth / Rth

11. Problem-Solving Flow for Exam Questions

Use this same order whenever a circuit looks complicated. It saves time and prevents random mistakes.

Exam Workflow

Fast Attack Order

  1. Mark known values and units clearly.
  2. Identify series parts, parallel parts, and obvious nodes.
  3. Simplify the circuit if possible.
  4. Use Ohm's law and divider rules first.
  5. If still complex, apply KCL or KVL.
  6. Check sign convention and answer units.

Common Mistakes

  • Adding parallel resistors directly instead of using reciprocals.
  • Mixing up same current and same voltage.
  • Forgetting that equivalent parallel resistance is smaller than the smallest branch.
  • Using peak instead of RMS in AC power questions.
  • Forgetting to convert mA to A, kOhm to Ohm, or hours to seconds.

One-minute recall sheet

  • Inductor steady DC -> short circuit.

  • Loop voltages: use KVL.
  • Output between two resistors in series: try voltage divider.
  • Branch current in parallel: try current divider.
  • Charging curve: think exponential and time constant.

Panic Mode

  • Need current? Use I = V / R or KCL if branches meet at a node.
  • Need voltage across one resistor? In series, use voltage divider. In parallel, each branch shares the same voltage.
  • Stuck on AC? Convert peak to RMS, then use impedance.
  • Need a graph idea? RC and RL are exponential, not linear.
